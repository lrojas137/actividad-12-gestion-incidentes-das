# Capacitación del equipo

El equipo de desarrollo será capacitado en el proceso de gestión de incidentes definido para el sistema de registro de pacientes.

La capacitación incluye los siguientes aspectos:

- Uso de GitHub Issues para registrar incidentes.
- Clasificación de incidentes mediante etiquetas.
- Priorización según el impacto del incidente.
- Uso del tablero de GitHub Projects para hacer seguimiento.
- Asignación de responsables.
- Documentación de la solución aplicada.
- Cierre del incidente después de las pruebas y validación.

Esta capacitación permite que el equipo conozca el flujo de trabajo y pueda atender los incidentes de forma ordenada, trazable y documentada.
